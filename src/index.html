Tentang Diri Saya

Halo, Saya Wahyudi Seorang Web Developer

Saya adalah mahasiswa semester akhir di universitas PGRI Wiranegara Bidang Studi Ilmu Komputer Semester 8 dengan IPS 3,80. Saya memiliki ketertarikan yang kuat terhadap pengembangan perangkat lunak. Selama masa studi, saya telah menguasai berbagai bahasa pemrograman seperti JavaScript, TypeScript, dan PHP. Saya memiliki pengalaman dalam mengembangkan aplikasi web menggunakan framework seperti React, Vue, dan Laravel.

Sebagai seorang yang antusias dalam belajar, saya senantiasa mengikuti perkembangan teknologi terbaru dan selalu berusaha meningkatkan keterampilan teknis saya. Saya juga aktif terlibat dalam proyek-proyek kelompok yang membantu saya untuk memperkuat kemampuan saya dalam bekerja secara kolaboratif, mengatasi tantangan teknis, dan mengelola waktu secara efektif.
